Drop your gallery media here. The landing page shows them in this order:

  demo.mp4         <- the VIDEO that plays first (a screen recording of Bloop Maker)
  demo-poster.png  <- a still image shown on the video before it plays (optional)
  shot1.png        <- screenshots, shown after the video
  shot2.png
  shot3.png
  shot4.png

Anything missing is skipped automatically, so you can add files over time.
To add more/rename, edit the MEDIA list near the bottom of landing/index.html.
Recommended: 16:10-ish images, video as .mp4 (H.264).
